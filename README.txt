# Secure EduCloud

Secure EduCloud is a Cloud-Based Document Management System tailored for primary schools. It allows admins and teachers to securely upload, manage, and retrieve documents, while students and parents can view important information like announcements and events.

## Technologies Used
- PHP
- MySQL
- HTML/CSS/JS
- GitHub for version control

## Features
- Role-based access (Admin, Teacher, Student)
- Document Upload, Edit, Delete
- Advanced Search Functionality
- Cloud Storage Integration (Local now, expandable to Firebase/S3)
- Chat and Announcement System

## How to Use
1. Clone this repository
2. Import `secure_educloud.sql` into your MySQL
3. Update your DB credentials in `includes/db.php`
4. Run project using XAMPP (htdocs)
